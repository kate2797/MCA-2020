Comparing findings from histograms

By looking at the histograms of the MFCC feature which commonly used for distinguishing between music instruments, we can clearly see that none of these tracks uses the same set of instruments since they all look a bit different. This makes sense as each track belongs to a different genre (rock, electronic, classical) and each genre uses a different set of instruments. The biggest outlier is the song Air which is a piano song. Although, in my opinion, Malala (rock) and Indust (electronic) are not very similar when you listen to them their MFCC histograms do not differ as much. This might be because besides other instruments, drums are fundamental to both genres.
